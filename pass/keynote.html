        전체 문항 리스트
        환경설정 및 단축키 안내
        초기화 주의
        폼을 재생성하면 현재 화면에서 입력 중이던 내용이 모두 사라지고 빈 폼으로 변경됩니다.
        • 새 문서: Ctrl + Alt + N
        • 저장하기: Ctrl + S
        • 문서 검색: Ctrl + E
        • 몰입 모드(UI 숨기기): Ctrl + H
        • 카드 모드(1문항씩 보기): Ctrl + M (좌/우 방향키로 이동)
        • 페이지 닫기: ` (백틱, 입력 중이 아닐 때)
        • 화면 스크롤: 화살표 위/아래 (입력 중이 아닐 때)
        • 글자 확대/축소: Ctrl + ➕ / ➖
        • 두음 강조(남색+흰글씨): Ctrl + Alt + M (에디터 포커스 시)
        • 두음 조회(사이드바 탭): Alt + I
        • 이미지 크게 보기(확대): 에디터에서 이미지 클릭 후 나타나는 툴바의 버튼 클릭
        • 에디터 서식: 취소선(Ctrl+0) | 빨강(Ctrl+Alt+R) | 파랑(Ctrl+Alt+B)
        • 서식 지우기: Ctrl + Shift + X
