유용한 단축키 & 팁

  • 새 문서: Ctrl + Alt + N
  • 저장하기: Ctrl + S
  • 글자 확대/축소: Ctrl + ➕ / ➖
  • 이미지 크게 보기: 에디터 내 이미지 더블클릭
  • 에디터 굵게: Ctrl + B
  • 두음 강조(남색+흰글씨): Ctrl + Alt + M
  • 에디터 서식: 취소선(Ctrl+0) | 빨강(Ctrl+Alt+R) | 파랑(Ctrl+Alt+B)
  • 컬러 형광펜: 주황(1) 파랑(2) 노랑(3) 녹색(4) 분홍(5) 갈색(6) 회색(7) *Ctrl과 함께
  • 표(Table) 지우기: 표의 맨 앞이나 뒤 클릭 후 백스페이스(지우기)
  • 이미지 강조(테두리): 에디터 내 이미지 클릭 시 나타나는 플로팅 메뉴 사용
  • 서식 지우기: Ctrl + Shift + X
