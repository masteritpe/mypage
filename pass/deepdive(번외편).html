    뷰 & 레이아웃 변경 (어디서나)
    • Alt + 1 : 에디터 모드
    • Alt + 2 : PDF 모드 (탐색창 숨김)
    • Alt + 3 : PDF 전체 모드 (탐색창 표시)
    • Alt + 0 / ESC : 기본 화면으로 복구
    • Ctrl + H : 몰입 모드 (문제 편집 상세)
    • Ctrl + M : 프리젠테이션 모드 (PDF 포커스)
    • Ctrl + Q : 리스트 모드 (문제 목록 위주)
    텍스트 서식 지정 (에디터 내)
    • Ctrl + B / I / U : 굵게 / 기울임 / 밑줄
    • Ctrl + 0 : 취소선 그기
    • Ctrl + 1 ~ 7 : 형광펜 칠하기 (다양한 색상)
    • Ctrl + Alt + R / B / G : 글자 색상 (빨/파/초)
    • Ctrl + Alt + M : 텍스트 블록 씌우기 (키워드 강조용)
    • Ctrl + + / - : 선택된 폰트 크기 확대/축소
    • Ctrl + Shift + X : 지정된 텍스트 서식 초기화
    시스템 및 탐색
    • Ctrl + S : 현재 작성중인 내용 저장하기
    • Ctrl + Alt + N : 새 문제 만들기
    • ↑ / ↓ (방향키) : 다음/이전 문제로 이동 (입력창 포커스 해제 시)
