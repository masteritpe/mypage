유용한 편집 단축키

  • 새 항목: Ctrl + Alt + N
  • 저장하기: Ctrl + S 또는 Ctrl + Enter
  • 글자 확대/축소: Ctrl + ➕ / ➖
  • 에디터 서식: 취소선(Ctrl+0) | 빨강(Ctrl+Alt+R) | 파랑(Ctrl+Alt+B)
  • 두음 강조(남색+흰글씨): Ctrl + Alt + M
  • 컬러 형광펜: 주황(1) 파랑(2) 노랑(3) 녹색(4) 분홍(5) 갈색(6) 회색(7) *Ctrl과 함께
  • 서식 지우기: Ctrl + Shift + X
  • 몰입 모드: H (포커스 밖에서)
  • 완료 체크: Space (포커스 밖에서)
  • 목록 이동: ⬅ / ➡ (포커스 밖에서)
