Node.js 서버 실행 가이드

1. 패키지 설치 (최초 1회)

npm init -y
npm install express cors puppeteer

2. 백엔드 서버 시작

node server.js

3. 서버 프로세스 강제 종료 (오류 시)

taskkill /F /IM node.exe
