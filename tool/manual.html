키보드 단축키 안내

공통 및 기호

  • 새 페이지 작성 Ctrl + Alt + N
  • 현재 문서 저장 Ctrl + S
  • ▶ (구분 추가) Alt + 1
  • ● (닷 추가) Alt + 2
  • - (리스트 추가) Alt + 3
  • → (화살표 추가) Alt + 4

에디터 서식

  • 굵게 / 기울임 / 밑줄 Ctrl + B / I / U
  • 취소선 적용/해제 Ctrl + 0
  • 글꼴 크기 확대/축소 Ctrl + + / -
  • 글자색 (빨/파/초) Ctrl + Alt + R / B / G
  • 마커 (7가지 색상) Ctrl + 1 ~ 7
  • 두음 강조 (남색 배경) Ctrl + Alt + M
  • 서식 지우기 Ctrl + Shift + X
