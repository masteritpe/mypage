대화 기록

관련 첨부 파일(최대 10개) 에디터 밖에서 붙여넣기 시 파일로 추가됨

단축키 및 편집 가이드

📄 문서 관리 및 화면 제어

  • 새 문서 생성: Ctrl + Alt + N
  • 현재 문서 저장: Ctrl + S
  • 몰입 모드 (메뉴 숨김): Ctrl + H
  • 선택 글자 확대/축소: 드래그 후 Ctrl + ➕ / ➖ (또는 Ctrl + 휠)

🖼️ 이미지 제어 팁

  • 이미지 크게 보기: 본문 이미지 더블클릭 (단일 클릭은 정렬/크기조절용)
  • 이미지 붙여넣기: 에디터 내부 클릭 후 Ctrl+V 시 본문에 바로 삽입
  • 사이즈 조절: 에디터 내 이미지 1회 클릭 시 나타나는 핸들로 조절
  • 정렬: 이미지 1회 클릭 시 상단에 나타나는 툴바(좌/중/우) 활용

✍️ 텍스트 서식 및 강조

  • 텍스트 굵게: Ctrl + B
  • 취소선 적용/해제: Ctrl + 0
  • 글자색 빨강: Ctrl + Alt + R
  • 글자색 파랑: Ctrl + Alt + B
  • 두음 강조 (남색배경+흰글씨+굵게): Ctrl + Alt + M
  • 모든 서식 지우기: Ctrl + Shift + X

🖍️ 컬러 형광펜 (배경색)

  • 주황 / 파랑 / 노랑: Ctrl + 1 / 2 / 3
  • 녹색 / 분홍 / 갈색 / 회색: Ctrl + 4 / 5 / 6 / 7

🖼️ 표(Table) 제어 팁

  • 표 지우기: 표의 맨 앞이나 맨 뒤를 클릭한 후 Backspace (백스페이스)
